# SkillSphere - Online Learning Platform

## Purpose
SkillSphere is a modern online learning platform where users can explore courses, view details, login/register, update profile information, and search courses by title.

## Live URL
https://skillsphere-sable.vercel.app/

## Key Features
- Responsive homepage with hero section
- Popular courses based on highest rating
- All courses page with search functionality
- Protected course details page
- Email/password authentication using Better Auth
- Google social login
- My Profile page
- Update profile name and image URL
- Toast notifications
- Loader and custom not-found page
- Persistent navbar and footer

## Tech Stack
- Next.js App Router
- TypeScript
- Tailwind CSS
- DaisyUI
- Better Auth
- Prisma
- SQLite
